# Mini Project for Node.js

### Starting MongoDB
```

Windows
1.1 cd /Users/xxx/Desktop/mongo/bin
1.2 mongod --dbpath . --storageEngine=mmapv1

2.1 cd /Users/xxx/Desktop/mongo/bin
2.2 mongo

macOS
1.1 cd /Users/xxx/Desktop/mongo/bin
1.2 ./mongod --dbpath .

2.1 cd /Users/xxx/Desktop/mongo/bin
2.2 ./mongo

```
